# Introduction

**DYP** (Dynamic Prompts Generator) is a CLI tool that renders interactive prompt templates via a Terminal User Interface (TUI). It lets you define reusable templates with placeholders that get filled through an interactive questionnaire.

## Why DYP?

- **Interactive**: Fill in prompts via a TUI instead of editing files manually
- **Template-based**: Write your prompt once, use it many times
- **Flexible**: Supports text inputs, dropdowns, and multi-selects
- **Portable**: Prompts are stored as simple Markdown files with YAML frontmatter

## How It Works

1. Create a Markdown file with `{{variable|type<options>}}` placeholders
2. Run `dyp` to select and render a prompt
3. Answer the interactive questions
4. Get your rendered output in the terminal

```
{{Name|text}} → Interactive text input
{{Role|select<developer,designer,manager>}} → Dropdown selection
{{Skills|multiselect<Go,Python,React>}} → Multi-select
```

## Quick Example

Create a prompt file `~/.prompts/hello.md`:

```markdown
---
title: "Hello Generator"
---

Hello, {{Name|text}}! You are a {{Role|select<Developer,Designer,Manager>}}.
```

Run:

```bash
$ dyp
```

Select "Hello Generator", answer the questions, and get your rendered output!

## Commands

| Command | Description |
|---------|-------------|
| `dyp` | Interactive prompt selector |
| `dyp list` | List all available prompts |
| `dyp render <filename>` | Render a specific prompt file directly |

## Options

- `--prompts <path>` - Custom prompts directory (default: `~/.prompts`)
- `--verbose` - Enable debug logging
