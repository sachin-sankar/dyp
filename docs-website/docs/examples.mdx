# Examples

Here are example prompts you can use with DYP. Copy these to your `~/.prompts/` directory.

## Available Examples

| Example | Description |
|---------|-------------|
| Business Proposal | Generate professional business proposals |
| Podcast Episode Script | Create podcast episode outlines |
| Project Status Report | Generate project status updates |
| Social Media Content Calendar | Plan social media content |
| Training Course Outline | Create course curricula |
| Product Launch Plan | Plan product launches |
| Marketing Email Campaign | Design email campaigns |
| Meeting Agenda Template | Create meeting agendas |
| Job Description | Write job postings |
| Academic Research Paper | Outline research papers |

## Business Proposal

```markdown
---
title: "Generate Business Proposal"
---

Create a business proposal for {{What is the project or service?|text}} to be presented to {{Who is the client?|text}}.

Proposal details:
- Proposal type: {{What type?|select<sales,partnership,grant,investment,service_agreement>}}
- Value: {{What is the value/proposal amount?|text}}
- Timeline: {{What's the expected timeline?|select<1_month,3_months,6_months,1_year,ongoing>}}

Client info:
- Industry: {{What industry is the client in?|select<technology,healthcare,finance,education,government,retail>}}
- Company size: {{How large is their company?|select<startup,small,medium,large,enterprise>}}

Include:
- Executive summary
- Problem statement
- Proposed solution
- Pricing breakdown
- Timeline and milestones
- Team overview
- Case studies or testimonials: {{Include how many?|select<1,2,3,5>}}
- Terms and conditions
- Next steps

Format: {{What format?|select<pdf,docx,slides,pdf_and_slides>}}
```

## Podcast Episode Script

```markdown

---

Segments:
1. Intro - {{Intro Duration|select<1_min,2_min,3_min,5_min>}}
2. Topic 1: {{Topic 1|text}}
3. Discussion: {{Discussion Points|multiselect<Guest Interview,Panel Discussion,Solo Segment,Audience Q&A>}}
4. Topic 2: {{Topic 2|text}}
5. Ad Read: {{Ad Read|select<yes,no>}}
6. Outro - {{Outro Duration|select<1_min,2_min,3_min,5_min>}}

Guest Info:
- Name: {{Guest Name|text}}
- Role: {{Guest Role|text}}
- Company: {{Guest Company|text}}

Key Takeaways: {{Key Takeaways|multiselect<3,5,7>}}
```

## Copying Examples

To use these examples, copy them to your prompts directory:

```bash
# Copy all examples
cp examples/*.md ~/.prompts/

# Or copy individual examples
cp examples/business-proposal.md ~/.prompts/
```

Then run:

```bash
dyp
```
